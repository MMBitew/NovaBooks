# Images Folder

Place your character images here:

## Required Images

1. **hagrid.jpg** - Photo of Hagrid character
2. **dumbledore.jpg** - Photo of Dumbledore character

## Recommended Specs

- **Size:** 150px wide × 200px tall (or any 3:4 aspect ratio)
- **Format:** JPG or PNG
- **File size:** Under 100KB each for fast loading

## Where to Find Images

1. **Google Images** - Search "Hagrid Harry Potter character" or "Dumbledore Harry Potter"
2. **Harry Potter Wiki** - https://harrypotter.fandom.com
3. **Official Warner Bros** - Movie stills and promotional photos

## How to Add

1. Download or create your images
2. Name them exactly:
   - `hagrid.jpg`
   - `dumbledore.jpg`
3. Place them in this folder
4. The app will automatically load them

**Note:** Until you add these images, the app will show purple placeholder boxes with the character names.
